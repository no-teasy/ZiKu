## 文源字体简介

文源字体基于思源字体进行二次开发，使之更加适合在简体中文情景下使用以及简体中文 TeX 排版领域的中文字体，开源免费商用。

文源字体支持的大多数汉字（GB 18030-2022 实现级别 2 范围内全部）采用中国大陆字形标准，且非汉字符号迎合简体中文一般使用习惯。以上字体均采用 OFL 协议，任何个人与团体均可免费商用。

### 字体特性

- **适合简体中文**：基于思源字体进行二次开发，更加适合在简体中文情景下使用
- **TeX 排版优化**：特别针对简体中文 TeX 排版领域进行了优化
- **广泛的字符支持**：支持 GB 18030-2022 实现级别 2 范围内全部汉字
- **符合中国大陆字形标准**：采用中国大陆字形标准，非汉字符号迎合简体中文一般使用习惯
- **免费商用**：采用 OFL 协议，任何个人与团体均可免费商用

### 适用场景

- 简体中文文档排版
- TeX 排版系统
- 学术论文和出版物
- 商业设计和印刷
- 网页和应用界面设计
