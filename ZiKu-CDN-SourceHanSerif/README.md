## 思源宋体简介

思源宋体（Source Han Serif）是由 Google 与 Adobe 联合打造的开源 Pan-CJK 字体，以其高质量字形、完整字重体系和免费商用特性，成为多场景设计的理想选择。

作为一款支持中日韩统一表意文字的泛 CJK 字体，它包含超过 50,000 个字符，覆盖 GB2312、GBK 等主流中文字符集，同时支持繁体中文、日文假名和韩文 Hangul。

### 字体特性

- **专业级设计品质**：由 Google 与 Adobe 联合打造，确保字体的高质量和一致性
- **完整字重体系**：提供 ExtraLight、Light、Regular、Medium、SemiBold、Bold、Heavy 等多种字重
- **广泛的字符支持**：包含超过 50,000 个字符，覆盖多种语言和字符集
- **免费商用**：采用 SIL Open Font License 1.1 协议，允许免费用于个人与商业项目

### 适用场景

- 书籍正文、网页主体内容
- 杂志内文、重点段落
- 小标题、导航菜单
- 主标题、强调文本
- 海报标题、封面设计
