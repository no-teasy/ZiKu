## 思源黑体简介

思源黑体（Source Han Sans，Google 称其为 Noto Sans CJK）是由 Adobe 与 Google 联手推出的开源字体，覆盖简体中文、繁体中文、日文和韩文，还包含了拉丁文、希腊文和西里尔文字形，真正实现了泛东亚语言的无缝排版支持。

这款字体的出现，为全球设计师提供了一款高质量且完全免费可商用的中文字体解决方案。每种字体粗细的总字形数达到了 65,535 个，这是 OpenType 格式支持的最大上限，而整个字体家族的字形个数接近 50 万个。

### 字体特性

- **广泛的语言支持**：覆盖简体中文、繁体中文、日文、韩文、拉丁文、希腊文和西里尔文
- **完整的字重体系**：提供从 ExtraLight（超细）到 Heavy（特粗）七种不同的字重选择
- **区域化优化**：针对不同地区的汉字标准进行了优化，提供符合中国、台湾、香港、日本和韩国地区标准的汉字版本
- **免费商用**：采用开源许可证发布，任何人都可以免费下载、使用甚至修改这款字体

### 适用场景

- 手机界面设计
- 电子书阅读器
- 网页设计
- 印刷品
- 多语言文档排版
