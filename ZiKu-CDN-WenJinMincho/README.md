## 文津宋体简介

文津宋体是一套以思源宋体及其衍生版本、利用 GlyphWiki 数据和修改版 Kage 引擎生成的汉字为主进行二次开发的大字符集宋体字体套装，实现 Unicode 定义的汉字和表意文字异体数据库（IVD）注册的字形全覆盖，其中 Unicode 汉字字形主要以中国大陆地区字形标准呈现。

该字体系列采用 OFL 协议，任何个人与团体均可免费商用。当今免费商用乃至开源的汉字大字符集字体相对较少，文津宋体的出现为用户提供了更多选择。

### 字体特性

- **大字符集支持**：实现 Unicode 定义的汉字和表意文字异体数据库（IVD）注册的字形全覆盖
- **符合中国大陆字形标准**：Unicode 汉字字形主要以中国大陆地区字形标准呈现
- **基于思源宋体**：以思源宋体及其衍生版本为基础进行二次开发
- **免费商用**：采用 OFL 协议，任何个人与团体均可免费商用

### 适用场景

- 对字符覆盖要求较高的排版场景
- 学术研究和出版
- 多语言混排文档
- 商业设计和印刷
